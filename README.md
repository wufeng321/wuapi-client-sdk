#SDK模块
```aidl
<groupId>com.wufeng</groupId>
<artifactId>wuapi-client-sdk</artifactId>
<version>0.0.1</version>
<name>wuapi-client-sdk</name>
```

使用该SDK，在你的代码中添加以下依赖：

```xml
 <dependency>
    <groupId>com.wufeng</groupId>
    <artifactId>wuapi-client-sdk</artifactId>
    <version>0.0.1</version>
 </dependency>
```
